# Rihan227 English-calc

## How do I install these formulae?

`brew install rihan227/english-calc/<formula>`

Or `brew tap rihan227/english-calc` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rihan227/english-calc"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
